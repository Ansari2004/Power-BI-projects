# Sales Dashboard Power BI Project

## Overview
This Power BI project provides a comprehensive sales dashboard that visualizes key sales metrics, including profit, quantity, payment modes, and monthly trends. The dashboard is designed to help analyze sales performance across different categories, customers, and time periods.

## Features
- **Sum of Profit**: Displays the total profit (37K).
- **Sum of Quantity**: Shows the total quantity sold (5615) with a breakdown by category.
- **Sum of Amount**: Highlights the total amount (438K).
- **Sum of Average**: Indicates the average sales value (121K).
- **Sum of Quantity by Category**: Pie chart illustrating the distribution of quantities across categories.
- **Count of PaymentMode by Category**: Donut chart showing the count of payment modes by category.
- **Sum of Profit by Category**: Bar chart comparing profit across categories (Electronics, Furniture, Clothing).
- **Sum of Profit by CustomerName**: Bar chart displaying profit distribution by customer names (Madan, Aarushi, etc.).
- **Sum of Profit by Month**: Bar chart tracking profit trends across months (January to December).

## Data Source
The data used in this project includes sales records with metrics such as profit, quantity, payment modes, customer details, and monthly sales figures.

## How to Use
1. Open the Power BI file (`Sales_Dashboard.pbix`) in Power BI Desktop.
2. Use the interactive filters (State and Quarter) to analyze data by specific regions and time periods.
3. Explore the visuals to identify sales trends and performance metrics.

## Installation
- Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
- Clone or download this repository.
- Open the `.pbix` file to view the dashboard.
